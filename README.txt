스테이지---------------
에너미 룸을 여러개 만들고
방을 배치할 때마다 에너미룸 카운트를 올린다
이 카운트를 이용하여 특수 방을 배치한다.

특수 방은
골드를 소모하는 무기 교체방,
함정이 많지만 유물을 획득하는 방
무료 석상 1개와 골드를 소모해 능력치를 올리는 석상이 있는 방
기본 석상은 3개로 이후 고고학자 npc를 통해 석상의 종류를 늘릴 수 있음.

기본 무기는 6개로, 나머지 무기들은 재료 조합을 통해 맵에서 스폰된다.


첫 스테이지보스를 처치할 때마다 특별한 조각을 얻게(애니메이션 보여주기)

버프 애니메이션:
플레이어를 중심으로
아우솔처럼 방패 아이콘 돌기 같이 버프 애니메이션 추가

애니메이션 컨트롤러를 추가하고

스프라이트 리스트를 버프스프라이트 스킬 버프스프라이트로 나눈다음 작업

NPC----------------------------------------------
메인 로비 NPC:

처음부터 존재하는 npc:
광고를 보여주는 npc
도서관 and 튜토리얼 npc(안경쓴 쿠키)

스테이지 클리어 시 구출되는 npc:
튜토리얼 클리어 시 스킬 상인(마법사 쿠키)를 구출
1단계 클리어 시 장인 토스트 npc를 구출(무기 조합 가능해짐)
2단계 클리어 시 고고학자 쿠키를 구출(돈을 지불하여 새로운 석상 개방)
