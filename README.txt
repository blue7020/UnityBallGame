스테이지---------------
에너미 룸을 여러개 만들고
방을 배치할 때마다 에너미룸 카운트를 올린다
이 카운트를 이용하여 특수 방을 배치한다.

상자 방 주변에는 함정을 배치해 놓는다.

특수 방은
골드를 소모하는 무기 교체방,
함정이 많지만 유물을 획득하는 방
무료 석상 1개와 골드를 소모해 능력치를 올리는 석상이 있는 방

인게임의 start 룸에서 로비의 npc 해금을 체크하며
아직 해금되지 않은 npc를 낮은 확률로 스타트룸에서 발견할 수 있게,

기본 무기는 6개로, 나머지 무기들은 재료 조합을 통해 맵에서 스폰된다.


첫 스테이지보스를 처치할 때마다 특별한 조각을 얻게(애니메이션 보여주기)
