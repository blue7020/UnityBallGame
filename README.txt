사용 유물과 패시브 유물을 만든 후
사용 유물은 UI에 띄워서 스킬 발동할 수 있게 하고
패시브 유물은 능력치가 적용될 수 있게 하기

사용과 패시브는 enum으로 구별 후, 사용 유물은 배열, 패시브 유물 리스트는 구별해서 적용

상점에서는 짝수 층마다 사용 유물만 팔기
(사용 유물은 배열이므로 플레이어가 현재 가진 사용 유물을 제외한 유물들은
계속 사용 유물 상점에 등장(3개) 일반 상점에서는 사용 아이템 1개 패시브 유물 2개 팜

1. 유물 스크립트를 만든다. -> 유물 컨트롤러를 만들어서 enum 으로 판단해
사용 유물은 배열로 일반 유물은 리스트에 넣는다
2. 보스 맵을 깨고 다음 스테이지로 넘어갈 때는 선택지를 준다
계속 하기, 그만 하기

계속 하기를 선택하면 현재 가진 유물을 모두 초기화 하겠습니까 라는 선택지를 준다

그만하기를 하면 해당 스테이지 만큼의 보상을 준다.

만약 도전 모드라면 현재 스테이지에서 유물이 모두 모이면 초기화 선택지만 준다.
초기화 시 추가 보상을 준다.
